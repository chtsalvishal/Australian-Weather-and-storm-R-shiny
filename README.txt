Steps to be followed while running this shiny application.
-----------------------------------------------------------

1) Make sure all the packages included in the R file is installed before running the application.

2) Check if the images and CSS files are available in the WWW folder.

3) Set the working directory of R studio to extracted file location and then open the R file.

4) Make sure the browser zoom is set to 80% in order for better experience.

5) If the icons on the Overall Mapview is absent, please restart your R session and repeat from step 1.

6) enjoy!
